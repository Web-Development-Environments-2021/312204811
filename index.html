About Me

ראיון קצר עם יוסף שרטו

שם: יוסף שרטו

תאריך לידה: 26.2.1935

יוסף שרטו נולד בכפר הונגורן שברמניה, ליעקוב וחיה ז"ל, שנרצחו בשואה
לאחר שהוריו נרצחו, הועבר יוסף לבית יותמים שם העביר את שנות נעריו.
בשנת 1947 החל יוסף שרטו את מסע העליה שלו לארץ ישראל, מסע שערך מספר חודשים רבים.

מה זכור לך מהעלייה עם הילדים היתומים לארץ ישראל?

יצאנו מרומניה בשנת 1947 לכיוון בולגריה, שם חיכו לנו 2 אוניות אמרקאיות בשם "פן יורק" ו"פרק קרשטתן".
את שתי האוניות העמיסו ב15,000 פליטים מרמוניה, הונגריה ובולגריה למיטב זיכרוני
משם יצאנו לכיוון ארץ ישראל, או בשמה המקורי פלשתינה.
לאחר מסע שארך כשבעה ימים החלו להתקרב לחופה של ארץ ישראל, אך נתקלנו ב7 אוניות
מלחמה בריטיות שלא אפשרו לנו להמשיך לכיוון ארץ ישראל.
האוניות שלנו החלו לנוע לאחר מכן לעבר קפריסין, שם חולקנו למחנות.
לצערי הרב, בשלב זה אני אחי ואחותי הופרדנו, ולא התראנו למשך יותר מ10 שנים.
לאחר כחודשיים בתוך מחנה הסגר, ובעקבות הסכם בין השלטונות הבריטים אפשרו לכל הילדים היתומים לעלות לארץ ישראל
וכך, לאחר מספר רב של חודשים, בשנת 1948 עליתי לארץ ישראל.

מה קרה איתך לאחר העליה?

מדינת ישראל קלטה אותנו, והעבירה אותנו ישר למשמפחות מאמצות.
למשפחה הראשונה קראו משפחת פיטל, והם גרו בכפר סבא. למשפחת פיטל היו 2 ילדים בוגרים, כאשר אני ושניהם עבדו יחד במשק החקלאי המשפחתי.
לאחר מספר חודשים הבן גויס לצבא ונהרג במלחת השחרור. האבל על המשפחה היה כבד והם החליטו לעזוב ואני נאלצתי לעבור למשפחה אחרת.
המשפחה שקלטה אותי היתה משפחת דובסטר, משפחה שעלתה מגרמניה לארץ.
למשפחה היום 2 בנים, אחד שוטר והשני היה עובד חקלאות, שלאחר מספר חודשים גם גויס לצבא.
לצערי, גם הוא נהרג במהלך שירותו הצבאי.
בשלב הבאה, בחברה המדינה להעביר את כל הילדים שעלו באותם השנים לקיבוץ דגניה ב' שם קיבלנו השכלה.
בקיבוץ למדנו במשך חצי יום, כאשר בחצי השני ההינו עובדים בעבודות המשק השונות.
בשנת 1956 התגייסתי גם אני לצה"ל, שירתתי בצה"ל במשך 5 שנים, ולאחר מכן ההיתי חייל מילואים.
השתתפתי במספר מלחמות ישראל בניהם: מלחמת סיני, מלחמת ששת הימים, מלחמת יום הכיפורים, ומלחמת לבנון הראשונה.
לאחר שהשתחררתי מהצבא, פגשתי את אישתי, ציפורה שלה אני נשוי עד היום.
יש לנו 3 ילדים, 12 נכדים ונינה אחת.

מה הם התחביבים האהובים עליך?

אני אוהב לבלות זמן בחצר, לפגוש חברים במועדון הגימלאים, לפגוש את בני המשפחה שלי.
בנוסף, אני מאוד אוהב לראות ספורט וכדורגל בעיקר. המועדון האוהב עלי הוא הפועל באר שבע.

Question 1: What are the three main diffrences between Web 1.0, Web 2.0 and Web 3.0?


  1. Communication
    In Web 1.0 there is no communication between the website and the clients, as Web 1.0 is a Read-Only platform.
    While Web 2.0 and Web 3.0 allows clients to communicate, Web 2.0 is a Read-Write
    Web 3.0 uses semantic methods where the computers uses "logic" and can analyzes data and provide wider knowledge.
  2. Data retrieval
    Read-Write platform
    Web 1.0 is the first stage of the World Wide Web, also known as WWW.
